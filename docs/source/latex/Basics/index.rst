********************
Βασικές Έννοιες
********************

Όπως έχουμε ήδη πει, η LaTeX είναι μία γλώσσα προγραμματισμού και όχι ένας επεξεργαστής κειμένου. Αυτό σημαίνει ότι για να μπορέσουμε να δουλέψουμε με αποδοτικό τρόπο θα πρέπει να μπούμε στη λογική του προγραμματισμού και όχι απλά πληκτρολόγησης κειμένου. Επίσης, δεν είναι αρκετό απλά να δημιουργήσουμε ένα αρχείο κειμένου στο οποίο προσθέσαμε και κάποιες εντολής, θα πρέπει να γνωρίζουμε και τις διαδικασίες επεξεργασίας αυτού του αρχείου, ώστε τελικά να παραχθεί το αντίστοιχο pdf αρχείο με το στοιχειοθετημένο κείμενο.

Στο παρόν κεφάλαιο θα εξηγήσουμε τις βασικές έννοιες που είναι αναγκαίες για την κατανόηση αυτού του τρόπου λειτουργίας. Μην ανησυχήσετε αν οι έννοιες αυτές σας φανούν αρχικά περίπλοκες ή έχουν μεγάλο όγκο πληροφορίας. Στο παρόν κεφάλαιο αλλά και σε όλη την έκταση του βιβλίου, έγινε προσπάθεια ώστε κάθε έννοια κατ' αρχάς να περιγράφεται με τον απλούστερο δυνατό τρόπο και επιπλέον να συνοδεύεται με όσο το δυνατό περισσότερα περιγραφικά παραδείγματα.
